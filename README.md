## Instalação

Baixe o script, vá até a pasta onde se encontra e dê as permissões para ele.

```sh
chmod +x script_dev.sh
```

Após isso, execute o arquivo:

```sh
./script_dev.sh
```

Bom, é basicamente isso, foi feito bem simples e prático. Aceito sugestões =)
